## 软件亮点

- 作品库用于统一管理新建作品、已有书籍和 TXT 导入内容
- 主工作区负责剧情点、章节编辑与写作流程推进
- 系统设置集中管理 Base URL、模型名和 API Key
- 适合长期整理小说、同人、系列文本或其他连载型内容

## 使用方式

1. 安装并打开软件后，优先进入系统设置完成模型配置。
2. 测试连接成功后，回到作品库创建新书或导入已有 TXT。
3. 打开主工作区，按章节和剧情点持续推进写作。
4. 需要调整时，随时返回作品库或系统设置继续编辑。

## 常见问题

### 为什么软件启动后不能直接写作？

通常是还没有先完成系统设置。请先填写模型服务地址、模型名和 API Key，再回到作品库或主工作区继续使用。

### 为什么作品库里没有内容？

首次使用时需要先创建新书，或者导入已有 TXT 大纲，作品库才会出现具体条目。
